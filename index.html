Skin Render

                    

                        
Quick, Easy, interactive 3D Skin Renders for Minecraft

                    

                    

                

            

            

            

                

                    

                        

                            

                                
Features


                                

                                    

                                        
    
                                            
  • Skins from PNG images (URLs or Base64 data)
    • 
                                            
  • Skins from usernames (via Crafatar)
    • 
                                            
  • Inner & Outer Skin Layers
    • 
                                            
  • Capes (can be any texture, user doesn't need to own the cape)
    • 
                                            
  • Skin & Cape data from MineSkin.org
    • 
                                        

                                    

                                    

                                        
    
                                            
  • All body parts's visibility can be toggled
    • 
                                            
  • All parts can be rotated/animated
    • 
                                            
  • Everything is rendered client-side (using THREE.js)
    • 
                                            
  • Pan, Zoom & Rotate controls, which can all be toggled individually
    • 
                                        

                                    

                                

                            

                        

                    

                

            

            

            

                

                    

                        

                            

                                

                                    

                                        

                                            
JavaScript


                                            

                                            

                                                
<script src="https://skinrender.ga/js/lib/OrbitControls.min.js"></script><!-- Required for interactive controls -->
<script src="https://skinrender.ga/js/render/skin.min.js"></script>

                                                
var skinRender = new SkinRender({/* options */}, document.getElementById("mySkinContainer"));
skinRender.render("inventivetalent");

About

                        

                            SkinRender provides really easy ways to add 3D Minecraft Skin Renders to your website.

                            

                            Either by directly adding the SkinRender script, which also provides customization options,

                            or by just inserting an IFrame embed, if you want a quick and uncomplicated solution with fewer customization options.
